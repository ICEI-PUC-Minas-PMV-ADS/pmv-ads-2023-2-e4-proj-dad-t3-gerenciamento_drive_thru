# Apresentação

Apresentamos um Software de Gerenciamento para Restaurantes Drive-Thru, oferecendo uma solução completa. Com interfaces intuitivas - um website para o restaurante gerenciar dados e produtos, e uma aplicação móvel para clientes administrarem suas informações e realizar pedidos de forma eficiente. Simplificamos o gerenciamento e a experiência do cliente, proporcionando eficiência e conveniência.

## DriveXpress - Software de gerenciamento de restaurantes DriveThru
